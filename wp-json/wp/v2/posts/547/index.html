\nWhat’s stayed the same in Office 2019?
\nEven though there is a different installation technology, you can still use the same software distribution tools, such as System Center Configuration Manager, that you currently use to deploy Office. Also, how you activate volume licensed versions of Office \u2013 for example, by using Key Management Service (KMS) \u2013 remains the same.<\/p>\n\n\n\n

The major version of Office 2019 is staying at 16.0, so some additional things are staying the same for those upgrading from Office 2016, including the following:
\nExisting Group Policy settings will continue to work, because the registry location for those settings will still be located in HKEY_LOCAL_MACHINE\\SOFTWARE\\Policies\\Microsoft\\Office\\16.0 and
\nHKEY_CURRENT_USER\\SOFTWARE\\Policies\\Microsoft\\Office\\16.0.
\nOther registry locations and settings information that reference 16.0 will still apply.
\nAdd-ins and other extensibility solutions that are compatible with Office 2016 will most likely be compatible with Office 2019 or will require only minimal testing.<\/p>\n\n\n\n
